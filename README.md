# 視線でカーソル操作＆ハンドジェスチャーでクリック操作
### （視線でカーソル操作＆まばたきでクリックの技術があるが、まばたきは生理現象なので、トリガーに使用するには現実的ではない）

## 実行コード

### mediapipeを使用する場合
~~~bash!
pip install mediapipe
~~~

### pyautoguiを使用する場合
~~~bash!
pip install pyautogui
~~~

### geze trackingを使用する場合
~~~bash!
pip install cmake
pip install dlib
pip install opencv-python
pip install gaze-tracking
~~~
